{{ get_svg_icon("lightbulb", "20", "#059669") }} ¿Cómo funciona?

1. Tokenización: El texto se divide en tokens (palabras o subpalabras).

2. Codificación: Cada token se convierte en números que el modelo entiende.

3. Embedding: El modelo neural transforma la secuencia en un vector denso.

4. Normalización: El vector se normaliza para facilitar comparaciones.

Resultado: Un vector de {{ embedding_dimensions }} dimensiones que captura el significado semántico del texto.
